#Instruções Básicas de Execução no LocalHost

1. **Banco de Dados:**
   - Abra seu gerenciador MySQL (XAMPP, DBeaver, etc).
   - Execute o arquivo `database.sql` anexo à entrega para criar a tabela `jogos` e popular os dados.
   
2. **Backend (API):**
   - Acesse a pasta `backend` pelo terminal.
   - Rode `npm install` para instalar as dependências (Express, MySQL2, Cors).
   - Configure as credenciais locais do seu banco de dados no arquivo `server.js`.
   - Rode `node server.js` para iniciar a API.

3. **Frontend:**
   - Com o backend rodando, abra o arquivo `index.html` (dentro da pasta `frontend`) no seu navegador ou utilize a extensão Live Server.
